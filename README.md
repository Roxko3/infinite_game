
Az SFW könyvtár felhasználásával készíts egy multithreadelt végtelen pályás játékprogramot, vizualizációval!

std névtér nem használható.

A programot az elkészülte után az órán prezentálni kell - legkésőbb az utolsón. (nem kell ppt vagy ilyesmi, csak bemutatni a projektornál), majd a canvasba kell feltölteni.

a) (2-es)

Pályageneráció.

Több szálon legyen képes a program egy pálya megadott részeit (chunk) generálni, illetve képes legyen ezen részek vizualizálására.

(Pl.: lehet 2d-s oldal, vagy felül nézetes tilemap, vagy akár valamilyen geometriai (vonalas) megoldás, vagy akár 3d-s terep / minecraft chunkok is.)

A generáció legyen thread safe.

b) (3-as)

Legyen egy karakter, aki tud menni a pályán.

Legyen ütközésvizsgálat.

Legyen valamilyen ellenfél (pl. ez lehet fal is).

Legyen pont számolva. (Nevet nem kell bekérni hozzá, menteni sem kell fájlba.)

c) (4-es)

Képes legyen a program új pályarészeket (chunkokat) dinamikusan generálni a játékos körül.

d) (5-ös)

A játék legyen kilépés nélkül újraindítható, legyen benne egy egyszerű menü, ahol mutatja az addigi legtöbb elért pontot.
